# Project Overview

This project demonstrates a Convolutional Neural Network (CNN) based image classification pipeline using grayscale images.

The objective of this assignment is to classify different image defect categories using deep learning techniques.

The notebook includes:

- Dataset loading
- Image preprocessing
- Image visualization
- CNN model building
- Model training
- Model evaluation
- Prediction visualization

---

# Dataset Information

The dataset contains grayscale images categorized into the following classes:

- normal
- scratch
- dent
- stain

Total images used:
- 480 images

Image size:
- 96 × 96 pixels
